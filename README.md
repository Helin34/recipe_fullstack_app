# 🍳 Recipe App

A modern full-stack recipe application where users can browse, search, create and delete recipes.

![Demo](./demo.gif)

---

## 🚀 Features

- 📋 List all recipes
- 🔍 Search recipes
- ⏱️ Sort recipes by cooking time
- ➕ Create a new recipe
- 🗑️ Delete recipes
- 📱 Responsive design
- 🔄 Real-time data updates with React Query

---

## 🛠️ Technologies

### Frontend

- React
- React Router DOM
- React Query (TanStack Query)
- Axios
- Tailwind CSS
- React Select
- React Toastify

### Backend

- Node.js
- Express.js
- UUID
- File System (JSON Database)
